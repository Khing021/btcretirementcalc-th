# Bitcoin Retirement Calculator (Thai Edition) 🇹🇭🚀

เครื่องคำนวณแผนการเกษียณด้วยบิตคอยน์ที่ปรับปรุงเพื่อคนไทย โดยอ้างอิงจากโปรเจกต์ต้นฉบับของ @pampeanodev

## 🤘 Vibe Coding Project
โปรเจกต์นี้ถูกพัฒนาขึ้นภายใต้จิตวิญญาณของ **"Vibe Coding"** — การสร้างสรรค์ซอฟต์แวร์ที่เน้นผลลัพธ์ที่จับต้องได้จริง การต่อยอดไอเดียอย่างรวดเร็ว และการทำงานร่วมกับ AI เพื่อส่งต่อเครื่องมือที่มีประโยชน์ให้กับชุมชนชาวส้มในไทย

## ✨ สิ่งที่เพิ่มเติมในเวอร์ชันนี้
- **ภาษาไทยเต็มรูปแบบ**: แปลทั้งหน้าจอ คำอธิบาย และตารางเป็นภาษาไทย
- **หน่วยเงินบาท (THB)**: ดึงราคาบิตคอยน์สดจาก Coinbase เป็นหน่วย THB พร้อมคำนวณมูลค่าล่วงหน้าเป็นเงินบาท
- **โมเดลการคาดการณ์หลากหลาย**: CAGR, S2F (Continuous Model), Power Law (Santostasi Formula), และ Michael Saylor 24 model
- **การใช้หน่วยที่ชัดเจน**: กำหนดหน่วยกำกับท้ายช่องกรอกข้อมูลชัดเจน (BTC / THB) เพื่อป้องกันความสับสน
- **โหมดการคำนวณ**: Conservative (ขายหมด) และ Optimized (HODL Mode)
- **เพิ่มเงินออมรายปี**: จำลองการปรับเพิ่มเงินออมรายปี (ตามเงินเฟ้อหรือเงินเดือนขึ้น) ในเดือนมกราคม
- **ตารางสรุปละเอียด**: ดูข้อมูลรายเดือน พร้อมคอลัมน์ "ซื้อบิตคอยน์เดือนละ (THB)" เพื่อติดตามกระแสเงินสด
- **Summary UI แบบใหม่**: สรุปข้อมูลสำคัญใน 2 คอลัมน์ เข้าใจง่าย พร้อมบอกเดือน/ปีที่เกษียณ และค่าใช้จ่ายเดือนสุดท้ายของชีวิตเกษียณ

# Bitcoin Retirement Calculator (Thai Edition) 🌍🚀
(English Version)

A specialized Bitcoin retirement planner for the Thai community, forked and enhanced from the original project by @pampeanodev.

## 🤘 Vibe Coding Project
This project was developed in the spirit of **"Vibe Coding"** — focusing on rapid iteration, creative flow using AI, and delivering immediate value to the local Bitcoin community.

## ✨ New in this Edition
- **Full Thai Localization**: UI, explanations, and data tables translated for Thai users.
- **THB Support**: Live price fetching from Coinbase in Thai Baht (THB) with future value projections in local currency.
- **Enhanced Models**: CAGR, S2F (PlanB Continuous), Power Law (Santostasi parameters), and Saylor 24 scenarios.
- **Clear Units**: Labeled inputs (BTC/THB) for better clarity.
- **Improved UI**: A new 2-column summary layout, precise retirement tracking, and "Monthly Buy (THB)" column.
- **Annual Savings Increase**: Simulate annual salary raises or inflation adjustments applied every January.

## 🛠 How to run
This project uses Vite and pnpm.
1. Clone: `git clone https://github.com/kssir/btcretirementcalc-th.git`
2. Install: `pnpm install`
3. Run: `pnpm run dev`
